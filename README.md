# Wahlmodul-Web-App

Interaktive Systeme Web-App zur Verbesserung des Wahlprozesses im Fachbereich Medien.

## Projektbeschreibung

Ziel des Projekts ist die Entwicklung einer benutzerfreundlichen Anwendung für Studierende und Lehrende.
Studierende können Module einfacher finden und filtern.
Lehrende werden bei der Erstellung von Angeboten unterstützt.

## Funktionen

### Studierende
- Filter über Tags
- Auswahl nach Studiengang
- Auswahl nach Semester
- Anzeige passender Module

### Lehrende
- 5-Schritte-Prozess
  - Zuordnung
  - Beschreibung
  - Wahlpflichtmodule
  - Organisation
  - Vorschau
